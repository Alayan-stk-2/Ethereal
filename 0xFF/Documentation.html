Ethereal Chess

Ethereal Chess is an chess engine developed during my highschool years and first year of university. This project aims to explore the various methods of game-tree search and evaluation, as well as a variety of topics such as; effective bit fields, cache friendly programming, multi-processing. Like any personal project, this project will never be completly finished, and I hope to continue to develop Ethereal Chess as I progress through University

Board Representation

Ethereal Boards are represented as a single integer array of length 256, while being visualized as a 16x16 array. The increased size allows an 8x8 standard chess board to be embedded inside the 16x16 array, and allows for bounds checkout with a single comparision, destination != WALL. The non embedded 8x8 portion of the array is all set to a value WALL.

Move Representation

All Moves are stored in a 32 bit unsigned integer, and are defined as follows

Move Type Bits 00-07 Bits 08-15 Bits 16-23 Bits 24-27 Bits 28-29 Bit 30 Bit 31
Normal From Sq To Sq Captured Type Move Type Castle Changes Turn Validated
Enpass From Sq To Sq Enpass Location Move Type None Turn Validated
Castle From Sq To Sq None Move Type Castle Changes Turn Validated
Promote From Sq To Sq Captured Type Move Type Promo Type (Flag) Turn Validated
